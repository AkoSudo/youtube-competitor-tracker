<task type="checkpoint:human-verify" gate="blocking">
  <name>Verify Phase 3: Ideas System</name>
  <what-built>
Complete ideas management system:
- ideas table in Supabase
- SaveIdeaModal for saving ideas with notes
- IdeasPage with filtering and deletion
- Real-time sync across teammates
- Nav badge showing idea count
  </what-built>
  <how-to-verify>
**Setup:** Run `npm run dev` and open app in browser

**Test 1: Save Idea Flow (REQ-IDX-001)**
1. Navigate to any channel with videos
2. Click "Save Idea" on a video
3. Verify modal shows video thumbnail and title
4. Try to save with < 10 char note - should show validation error
5. Enter a valid note (10+ chars) and your name
6. Click Save - modal closes, success toast appears
7. Your name should be remembered for next save

**Test 2: Ideas Page Display (REQ-IDX-002, REQ-IDX-003)**
1. Navigate to Ideas page
2. Verify saved idea appears with:
   - Full note text
   - Video thumbnail (clickable to YouTube)
   - Video title
   - Channel name
   - Your name and relative date

**Test 3: Delete Idea (REQ-IDX-004)**
1. Click delete button on an idea
2. Confirm in dialog
3. Idea should disappear, success toast

**Test 4: Real-time Sync (REQ-IDX-005)**
1. Open app in two browser windows
2. In window 1: Save a new idea
3. In window 2: Idea should appear within 2 seconds
4. In window 2: Toast notification should appear
5. Delete in window 1: Idea should disappear in window 2

**Test 5: Search Filter (REQ-IDX-006)**
1. Have 2+ ideas saved
2. Type part of a note in search box
3. List should filter to matching ideas
4. Type part of a video title
5. List should filter to matching ideas
6. Clear search - all ideas return

**Test 6: Channel Filter (REQ-IDX-007)**
1. Have ideas from 2+ different channels
2. Click channel dropdown
3. Should show only channels that have ideas
4. Select a channel
5. Only ideas from that channel should show
6. Select "All Channels" - all ideas return

**Test 7: My Ideas Toggle (REQ-IDX-008)**
1. Have ideas from different users (or simulate by editing added_by in Supabase)
2. Toggle "My Ideas" on
3. Only ideas matching your localStorage name should show
4. Toggle off - all ideas return

**Test 8: Nav Badge**
1. With ideas saved, check Nav shows count badge
2. Delete all ideas, badge should disappear or show 0
  </how-to-verify>
